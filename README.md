# 🌌 Poetry Platform

A simple poetry sharing web app where users can write, read, like, and explore poems with categories, views, and featured ranking.

---

## ✨ Features

- 📝 Create and publish poems  
- 📚 Browse poems by categories  
- ❤️ Like / Unlike poems  
- 👁 View counter for each poem  
- 🏆 Featured Top 7 poems (auto updated by likes)  
- 👤 User login system  
- ❤️ Favorites section  
- 📱 Responsive design (mobile + desktop)

---

## 🚀 Tech Stack

- Next.js (App Router)
- React.js
- Supabase (Database + Auth)
- Tailwind CSS
- GSAP (basic animations)
